Sobre el proyecto

¿Por qué no ir más allá de una simple App y diseñar un proyecto integral contra la violencia de género? Una idea innovadora, global y colaborativa que ayude a ponerle fin.

El proyecto Ni 1 Más se apoya sobre cuatro pilares: Prevención, asistencia, análisis y concienciación.

  • Prenvención: Notas, la aplicación para enviar alertas de forma rápida a la policía y a sus contactos.
  • Asistencia: Contactos, la aplicación que usarán los conocidos de la potencial víctima para conocer el estado de la misma.
  • Análisis: Hypatia, la plataforma de datos para realizar análisis de datos y obtener información interesante.
  • Concienciación: Semáforo UA, el test para evaluar cuál es la situación de una pareja, tanto para él como para ella.
